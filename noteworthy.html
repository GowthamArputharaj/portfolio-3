Project Details
//
The illustrations were done on the iPad Pro with the Apple Pencil, using Adobe’s powerful vector drawing tool Adobe Draw. After the base was done, I imported all illustrations into Illustrator and added subtle textures to the shadows and midtones.
Noteworthy
B-Side
//
The initial direction for the project was to create modern layouts that resemble classic or vintage sports playing cards. Even though I ended up going for a desaturated look in the end, these layouts still made the B-side of the project.
Noteworthy
Feature
//
The project was later featured in the Adobe Illustrator Draw section on Behance, alongside many inspiring illustration projects created on the iPad.
